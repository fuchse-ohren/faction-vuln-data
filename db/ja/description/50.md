Web アプリケーションは、パラメータ値を使用して、
クライアントがリダイレクトされるページのアドレス -- for
例: `yoursite.com/page.asp?redirect=www.yoursite.com/404.asp`

An
未検証のリダイレクトは、クライアントが
リクエスト内のパラメータ値に影響を与えるため、場所を制御します
リダイレクトの。たとえば、次の URL
「yoursite.com/page.asp?redirect=www.anothersite.com」は次の場所にリダイレクトされます。
「www.anothersite.com」。

攻撃者はこれらを悪用します
ユーザーを攻撃するソーシャル エンジニアリング攻撃の脆弱性
知らずに悪意のある Web サイトにアクセスしてしまう。

このツールが発見したのは、
サーバーはリダイレクト前にパラメータ値を検証しません。
クライアントを注入された値に変換します。
