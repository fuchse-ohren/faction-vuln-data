Basic Access Authentication は、base64 エンコードを使用する認証方法です。
そして、サーバーとクライアントの間でログイン認証情報をクリアテキストで送信します。これは
認証情報は暗号化されていないチャネルで送信されるため、安全ではないと考えられます
そのため、ネットワーク盗聴攻撃に対して脆弱になります。さらにbase64
エンコードは簡単にデコードされて元の情報が明らかになるため、弱いと考えられます。
