Web アプリケーション内から特定のアクションを実行するには、
オペレーティング システムのコマンドを実行し、
これらのコマンドの出力は Web アプリケーションによってキャプチャされ、返されます。
クライアントに。

OS コマンド インジェクションは、ユーザーが入力したときに発生します。
適切なサニタイズなしでこれらのコマンドのいずれかに挿入され、
その後、サーバーによって実行されます。

攻撃者はこれを悪用します
サーバー上で独自の任意のコマンドを実行する弱点。これ
単純な「ping」コマンドから内部マッピングまであらゆるものを含めることができます。
ネットワークを介してサーバーの完全な制御を取得します。



特定の操作の実行を注入して検証することが可能でした
適切な入力サニタイズが行われていないことを示すシステム コマンド
発生している。
