HTTP プロトコル自体はクリア テキストです。つまり、
HTTP 経由で送信されるメッセージをキャプチャしてコンテンツを表示できます。に
データをプライベートに保ち、傍受されるのを防ぎます。HTTP は多くの場合、
Secure Sockets Layer (SSL) またはトランスポート層のいずれかを介してトンネリングされる
セキュリティ (TLS)。これらの暗号化標準のいずれかを使用すると、
HTTPSと呼ばれます。
