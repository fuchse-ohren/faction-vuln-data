機密情報漏洩の脆弱性は、アプリケーションまたはシステムが意図せずに機密データを権限のない個人または団体に公開した場合に発生します。この脆弱性はさまざまな形をとる可能性があり、個人データ、パスワード、財務詳細、企業独自の情報など、さまざまな種類の機密情報が関与する可能性があります。

アプリケーションまたはシステムが、クレジット カード、パスワード、その他の機密データなどの機密情報を適切な保護なしで画面に表示すると、ショルダー サーフィンに対して脆弱になります。近くにいる人が画面をちらっと見て、適切な許可なしに機密情報にアクセスする可能性があります。

この問題は、インターフェイスが画面上のデータをマスクしている場合でも、アプリケーションが機密フィールドの完全な内容を Web インターフェイスに返す場合にも発生する可能性があります。たとえば、サーバーは保存されたクレジット カード番号全体を Web インターフェイスに返し、JavaScript またはその他のマスキング技術を通じてクレジット カード番号の一部のみを表示することができます。この場合、機密情報を完全に返す理由はなく、攻撃者がユーザーのアカウントに不正にアクセスできた場合、ユーザーをさらに侵害するために使用される可能性のある追加情報が開示されることになります。  :w
